CalculateCheapestPath
Программа для вычисления маршрута с наименьшими денежными затратами
Автор
Кузьмин Данила

Программа позволяет вычислять самый дешевый маршрут из первого города в город N при условии, что на преодоление одной дороги требуется задаваемое для каждой дороги определенное количество литров бензина, цена которого задается в городах.

Формат входных данных

На вход программы подается 2 файла формата .csv, первый содержит стоимости 1 литра бензина в городах, а второй содержит матрицу дорог.

Cодержание выходного файла

Выходной файл представляется в виде одного текстового файла, содержащего результат вычисления. В случае ошибки во входных данных, сообщение об ошибке выводится в консоль, выходной файл при этом не создается
